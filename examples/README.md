# UniSecure Examples

This directory contains example code and use cases for UniSecure.

## Examples Overview

### Code Security Examples
- `vulnerable_code.py` - Sample code with intentional vulnerabilities for testing
- `scan_codebase.py` - Example of scanning a codebase programmatically

### Application Security Examples
- `scan_webapp.py` - Example of scanning a web application

### Host Security Examples
- `scan_system.py` - Example of performing host security assessment

### Container Security Examples
- `scan_containers.py` - Example of scanning Docker container images

## Running Examples

```bash
# Install UniSecure first
pip install -e ..

# Run individual examples
python vulnerable_code.py
python scan_codebase.py
python scan_webapp.py
python scan_system.py
python scan_containers.py
```
